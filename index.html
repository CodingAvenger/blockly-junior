Blockly Junior

Here you can find some icons that can be used in Blockly, the web-based, graphical programming editor.

The purpose of these icons is making Blockly more easy for young children that have not learned to read yet (3-5 year-olds). It is similar to ScratchJr, which is a simpler version of Scratch.

Blockly

The project page is https://developers.google.com/blockly/
